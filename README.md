# 📖 About

**Store4Tech** is an online shopping platform designed for selling technology products such as laptops, smartphones, PC components, gaming accessories, monitors, keyboards, mice, headphones, and other electronic devices. This is also a final project developed for the Web Programming course at University of Transport Ho Chi Minh City (UTH).

The project aims to build a modern and responsive technology e-commerce website using PHP and MySQL while applying fundamental web development concepts and best practices.

The project focuses on:
- Clean and modern UI/UX
- Responsive design
- User authentication and authorization
- Product management
- Shopping cart and order processing
- Admin dashboard with full CRUD functionality
